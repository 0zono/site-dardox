# Birthday Ana

Página estática mobile-first para uma carta digital de aniversário.

## Publicação no GitHub Pages

Copie estes arquivos para o repositório `0zono/site-dardox` e publique a branch
configurada no GitHub Pages:

- `index.html`
- `styles.css`
- `script.js`
- `assets/birthday-bg.png`

Não há etapa de build nem dependências externas.
